This page demonstrates that all reveal.js plugins work correctly when embedded inline (without an iframe).

{{< slides src="slides/reveal-demo" height="600px" >}}

## Features Demonstrated

1. **Markdown Plugin** - Full markdown support in slides
2. **Highlight Plugin** - Syntax highlighting for code blocks
3. **Notes Plugin** - Speaker notes (press S)
4. **Zoom Plugin** - Alt+click to zoom
5. **Vertical Slides** - Navigate with ↑↓ arrows
6. **Fragment Animations** - Step-through content
7. **Overview Mode** - Press O to see all slides
8. **Custom Backgrounds** - Colors and images
